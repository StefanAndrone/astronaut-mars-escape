
This project is going to be a point and click game.

Storyline:

An astronaut gets on Mars, with his spaceship, by mistake. There, spaceship loses power. First scene is the astronaut talking to the ship AI, then the game begins.

The astronaut has to find a proper battery, but to do that he will have to face aliens.

The game will be Inkagames-like, as in in the style of Inkagames.
